# OS Project 3

## Running the Project

### Option 1: Directly

Run it directly 

```bash
python3 project3.py create file.idx
```

### Option 2: As a script

Make the entry point `project3.py` executable:

```bash
chmod +x project3.py
```

```bash
./project3.py create file.idx
```

## Seeing what you can do

for the whole tool:

```bash
./project3.py -h
```

or, for a specific subcommand:

```bash
./project3.py insert -h
```

Similarly, if you choose to run it directly.

```bash
python3y project3.py -h
```

or, for a specific subcommand:

```bash
python3 project3.py insert -h
```
